What is Capture the CTF (CTCTF)?

It is a week-long online competition is built by high school students, for high school students. this CTF offers a great challenge for veteran hackers, while still being really fun for beginners to get their feet wet. From cryptography to brute-forcing through algorithmic problems, from completing some beginner HTTP request problem to cracking the code to one of the hardest, CTCTF is going to be a great experience.
